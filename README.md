## research

Each directory in this repo is a separate research project carried out using AI tools. Every line of text was written by an LLM. Some projects contain reports from multiple models (Claude, ChatGPT, Gemini, Perplexity) for diversity of opinion and spotting commonalities. Sometimes I will also create a "unified" report.

Inspired by [Simon Willison's research repo](https://github.com/simonw/research).

## table of contents
### [Write-Ahead Logging](write-ahead-logging/) (2026-04-07)

Deep research reports on Write-Ahead Logging (WAL) — its theory, the ARIES algorithm, and concrete implementations in SQLite and PostgreSQL — generated independently by Claude and ChatGPT for comparison.
